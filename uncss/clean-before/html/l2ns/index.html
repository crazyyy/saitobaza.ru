Наш сайт посвящен анонсам серверов Lineage 2. На сайте не будет рейтингов серверов, не будет голосований, только анонсы про новые сервера Lineage 2. Для чего нужны анонсы серверов? Сайт с анонсами новых серверов – сэкономленное время для игрока. Время, которое не будет потрачено на поиск серверов с низкими рейтами или высокими, pvp серверов или классики Lineage 2. Каждый опытный игрок знает, как неприятно, когда тратишь огромное количество времени на прокачку, на крафт, на клан, а через месяц вайп. На нашем сайте – l2servers.org Вы найдете анонсы по серверам всех хроник: Interlude, High Five, Freya, Gracia Epilogue, Goddess of Destruction, Gracia Final. В каждой из хроник, есть свои плюсы и минусы. В анонсах новых серверов Вы узнаете, в чём они заключаются. Так как серверов с высокими и низкими рейтами открывается огромное количество, только наш проект поможет выбрать Вам нужный сервер. Если pvp сервер, можно с лёгкостью поменять на другой, то сервера с низкими рейтами будет тяжело бросать. В анонсах серверов л2 можно прочитать подробное описание сервера lineage 2 interlude, его особенности, узнать об изменениях, внесённых Администрацией.
