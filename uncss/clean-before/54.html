Плавная смена фонового изображения иил цвета

Сайтобаза / Блог / Плавная смена фонового изображения иил цвета

На одном из последних проектов нужно было создать эффект плавной смены/перехода background-image при наведении мышки на блок. Задать смену цвета в css3 можно без проблем, для этого используется css transition. А вот плавная смена фонового изображения — уже другая задача, стандартными средствами, казалось бы, этого не сделать.

Но на помощь приходит смекалка. Ниже я выложил код html и css с рабочим примером, а сейчас опишу порядок действия:

  1. Имеется блок, у которого есть два бекграунда — обычный, и при наведении
  2. Создаем внутри блока пустую пару span
  3. Для span внутри задать что это блочный элемент, который имеет размеры 100% на 100% от родительского
  4. Этот span получается перекрывает нам весь блок, потому добавляем ему backround-image родительского блока в обычном состоянии, а родительском блоку ставим за основу фоновую картинку, которая должна появляться при наведении
  5. Для span с помощью transition opacity задаем скорость смены прозрачности, например 1 секунду
  6. Для span блока при наведении на родительский элемент задаем прозрачность 0

Ну и логика работы — span просто перекрывает своим фоновым изображением картинку родительского элемента. При наведении на родителя — span блок (и его фон) становится постепенно прозрачным, что позволяет нам увидеть фоновое изображение родителя.

Вот и все, очень просто.

Накидал тут пример, правда ещё не очень разобрался с этим сервисом (был приятно удивлен поддержкой Emmet), потому вместо фоновой картинки сделал фоновый цвет, но думаю и так понятно что работать будет отлично.
