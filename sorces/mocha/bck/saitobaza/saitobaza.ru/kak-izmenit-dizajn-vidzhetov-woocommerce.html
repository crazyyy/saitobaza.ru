Как изменить дизайн виджетов WooCommerce

Продолжая настраивать плагин магазина для WordPress WooCommerce я столкнулся с тем, что нужно задать свой дизайн для некоторых виджетов. Точней ситуация была в следующем — дизайн главной страницы сайта содержал множество разных блоков — Новые товары, Лучшие товары, Товары со скидкой. Мне не очень хотелось переписывать плагин WooCommerce под себя, да и нужно было оставить клиенту возможность беспрепятственно обновляться. Поэтому решил выводить каждый блок отдельным виджетом.
Для этого каждый блок я подключил через отдельный sidebar, и в админке WP в разделе виджетов вывел нужные виджеты в назначенные для них места.
Естественно стандартный дизайн для таких блоков ну никак не подходил, так как клиент рисовал дизайн сам и не руководствовался какими-то нормами — делал как ему нравилось. Ну да и ок.
Поискав немного, наткнулся на замечательную статью некоего justin How to Override WooCommerce Widgets. Братюня предлагает 6 разных вариантов как задать свой дизайн виджетов. Я провел «полевые» испытания, несколько раз уронил WP и остановился на п.5 как самом надежном и самом лёгком.

Описываю пошагово как задать свой дизайн для виджетов WooCommerce

  • Создать папку «widgets» в корне вашей темы
  • Из папки «widgets» плагина скопировать все (или только нужные, если вы уже разобрались что вам нужно) файлы. Называются они «widget-название функции.php»
  • Отключить с помощью функции «unregister_widget» в файле functions.php вашей темы виджет плагина и подключить новый виджет, который мы скопировали к себе в тему. Об этом ниже
  • Редактируем новый виджет
  • ….
  • profit!

По третьему пункту — что бы отключить стандартный виджеи подключить свой, нужно в functions.php добавить функцию 

/* change woocemmerce onsale*/ 
add_action( 'widgets_init', 'override_woocommerce_widgets', 20 );
function override_woocommerce_widgets() {
  unregister_widget( 'WooCommerce_Widget_On_Sale' );
  register_widget( 'Davidoff_WooCommerce_Widget_On_Sale' );
}
include_once( 'widgets/widget-onsale.php' );

где WooCommerce_Widget_On_Sale — название стандартного виджета, а Davidoff_WooCommerce_Widget_On_Sale — мой вариант и путь к файлу моего виджета widgets/widget-onsale.php (относительно файла functions.php, естественно)
Название функции стандартного виджета (в нашем случае WooCommerce_Widget_On_Sale) мы можем посмотреть (а заодно и переименовать его) в файле widgets/widget-onsale.php нашей темы.

Собственно все, теперь можно редактировать widget-onsale.php по своему усмотрению.

Хе. Добавлю ещё. Если нужно подключить 2 и больше виджета, просто переименовываете функцию override_woocommerce_widgets на что-то типа override_woocommerce_widgets-2 и добавляйте 

add_action( 'widgets_init', 'override_woocommerce_widgets-2', 20 );
function override_woocommerce_widgets-2() {
  unregister_widget( 'WooCommerce_Widget_On_Sale' );
  register_widget( 'Davidoff_WooCommerce_Widget_On_Sale' );
}
include_once( 'widgets/widget-onsale.php' );
